# 在 Mr.🆖 AI 中使用 Mr.🆖 AI Hub

<Image
  alt={'在 Mr.🆖 AI 中使用 Mr.🆖 AI Hub'}
  cover
  src={'https://privacy.mr5ai.com/MrNG-AIHub.png'}
/>

本文档将指导你如何在 Mr.🆖 AI 中使用 [Mr.🆖 AI Hub](https://api.mr5ai.com/):

<Steps>

### 步骤一：获取 Mr.🆖 AI Hub API 密钥

- 注册一个 [Mr.🆖 AI Hub 账户](https://api.mr5ai.com/register)，你需要使用邮箱进行注册；
- 注册完毕后，前往 [API 金鑰](https://api.mr5ai.com/panel/token) 页面，点击 `新建 API 金鑰` 创建新的 API Key:

<Image
  alt={'打开创建窗口'}
  inStep
  src={
    'https://privacy.mr5ai.com/Open-Creation-Window.jpg'
  }
/>

- 创建 API Key

<Image
  alt={'创建 API Key'}
  inStep
  src={
    'https://privacy.mr5ai.com/Create-API-Key.jpg'
  }
/>

- 複製 API Key

<Image
  alt={'複製 API Key'}
  inStep
  src={
    'https://privacy.mr5ai.com/Copy-API-Key.jpg'
  }
/>

<Callout type={'warning'}>账户注册后，你有 69,000 免费额度。</Callout>

### 步骤二：在 Mr.🆖 AI 中配置 Mr.🆖 AI Hub

- 访问 Mr.🆖 AI 的`设置`界面
- 在`语言模型`下找到`Mr.🆖 AI`的设置项

<Image
  alt={'貼上 API 密钥'}
  inStep
  src={'https://privacy.mr5ai.com/Enter-API-Key.jpg'}
/>

- 貼上已複製的 API 密钥
- 为你的 AI 助手选择一个 AI 模型即可开始对话

<Image
  alt={'选择 AI 模型并开始对话'}
  inStep
  src={'https://privacy.mr5ai.com/Select-AI-Model.jpg'}
/>

<Callout type={'warning'}>
  在使用过程中你可能需要向 API 服务提供商付费，请参考 Mr.🆖 AI Hub 的相关费用政策。
</Callout>

</Steps>

至此你已经可以在 Mr.🆖 AI 中使用 Mr.🆖 AI Hub 提供的 AI 模型进行对话了。
